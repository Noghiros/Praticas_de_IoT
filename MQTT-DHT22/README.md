
# 📡 Pratica 2 - Monitoramento com DHT22 e Envio via MQTT (MicroPython)

Este projeto realiza a leitura de temperatura e umidade com o sensor **DHT22** e envia os dados para um **broker MQTT**.

---

## 🧰 Tecnologias e Componentes

- ESP32
- Sensor DHT22
- MicroPython
- Protocolo MQTT

---

## 🌐 Configuração MQTT

- **Broker:** broker.mqttdashboard.com  
- **Tópico:** Stefano-Calheiros-Stringhini  
- **Client ID:** teste.utfpr.edu.br

## 🔌 Conexões

| DHT22 | ESP32 |
|------|--------|
| VCC  | 3V3    |
| GND  | GND    |
| DATA | Pino 15|
