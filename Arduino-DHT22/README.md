# 🌡️ Pratica 1 - Leitura de Temperatura e Umidade com DHT22 (Arduino)

Este projeto realiza a leitura de temperatura e umidade utilizando o sensor **DHT22**, exibindo os dados no monitor serial.

---

## 🧰 Componentes

- Arduino (Uno ou similar)
- Sensor DHT22
- Jumpers

---

## 🔌 Conexões

| DHT22 | Arduino |
|------|--------|
| VCC  | 5V     |
| GND  | GND    |
| DATA | Pino 2 |
