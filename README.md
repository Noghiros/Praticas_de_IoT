# 🌡️ Praticas_de_IoT - DHT22 (Arduino + ESP32/MQTT)

Projetos desenvolvidos para a disciplina de Internet das Coisas (IoT), utilizando o sensor DHT22 para leitura de temperatura e umidade.

## 📦 Projetos

### 🔹 Pratica 1 - Leitura com Arduino
- Leitura de temperatura e umidade com DHT22
- Exibição via Serial Monitor

📁 `Arduino-DHT22/README.md`

---

### 🔹 Pratica 2 - Envio via MQTT (MicroPython)
- Conexão Wi-Fi
- Leitura do sensor DHT22
- Envio de dados para broker MQTT

📁 `MQTT-DHT22/README.md`
